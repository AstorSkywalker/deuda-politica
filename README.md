# Deuda Politica

This repository contains a static dashboard for exploring Honduras EG2025 presidential and municipal results in a single interface.

Published dashboard:

https://astorskywalker.github.io/deuda-politica/

The main dashboard is implemented in [index.html](./index.html) and is designed to open directly in a web browser on desktop or mobile devices.

Included data files:

- `jrvs_faltantes_presidencial_eg2025.csv`
- `jrvs_faltantes_municipal_eg2025.csv`
- `resultados-presidenciales-eg2025.xlsx`
- `resultados-municipales-eg2025.xlsx`

The dashboard includes:

- Integrated presidential and municipal views
- Summary indicators and comparison cards
- Filters for exploring results by territory and category
- Tables and visual summaries for quick review

To view it locally, open `index.html` in a browser.
